# MESCC

## DESCRIPTION

MESCC is a C compiler for the Z80 microprocessor on CP/M.  

## LIBRARIES

```c
    #include <mescc.h>     // Runtime. All programs must include this file first.
    #include <alloc.h>     // Memory allocation functions.
    #include <atexit.h>    // Code for atexit() function.
    #include <bsearch.h>   // Binary search function.
    #include <clock.h>     // Date & time functions for CP/M Plus.
    #include <conio.h>     // Console I/O functions.
    #include <cpm.h>       // CP/M specific functions.
    #include <ctype.h>     // Test and character conversion functions.
    #include <fileio.h>    // Stream file I/O functions.
    #include <fprintf.h>   // Formatted output to file.
    #include <mem.h>       // Memory functions.
    #include <printf.h>    // Formatted output to console.
    #include <qsort.h>     // Sort function.
    #include <rand.h>      // Pseudo-random number generation.
    #include <setjmp.h>    // Non-local jumps.
    #include <sprintf.h>   // Formatted output to memory.
    #include <stdbool.h>   // Boolean type and values.
    #include <string.h>    // String functions.
    #include <xprintf.h>   // Support library for printf, fprintf and sprintf.
    #include <z80.h>       // Z80 specific functions.
```
